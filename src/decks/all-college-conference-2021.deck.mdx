pick any concept --- work in groups, hands on, laptops out

# group formation

groups of 2--3, along the music/visuals axis

## discussion questions

1. which concept did you choose, and how did the creative angle shape that?
2. what kind of task did you set --- convergent or divergent?
3. how would you assess it?
4. what scaffolding does a College IT context need?
5. was it _fun_?
