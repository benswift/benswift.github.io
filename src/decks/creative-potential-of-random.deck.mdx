# a little history

![bg right:45%](./assets/creative-potential-of-random/bg-history.avif)

musikalisches würfelspiel --- 18th-century dice music

aleatoric composition --- 20th century

chance has always been an artistic tool

---

{/* _class: impact */}

not all randomness is created equal

(but this **isn't** a maths lecture)

---

![bg right:45%](./assets/creative-potential-of-random/bg-uniform.avif)

## uniform

every value in the range equally likely

`random(min, max)`

the workhorse

---

![bg right:45%](./assets/creative-potential-of-random/bg-gaussian.avif)

## gaussian

clustered around a middle, rarer at the edges

`randomGaussian()`

the bell curve

---

![bg right:45%](./assets/creative-potential-of-random/bg-discrete.avif)

## discrete

a fixed set of outcomes, weighted how you like

`random([a, b, c])`

dice, coins, choices

---

## weighting, the cheap way

```js
const loaded = [1, 2, 2, 2, 2, 3, 4, 5, 6];
const roll = random(loaded);
```

repeat the values you want more often

---

## weighting, the deliberate way

```js
if (random(100) < 10) {
  // the 10% branch
} else {
  // the other 90%
}
```

---

![bg right:45%](./assets/creative-potential-of-random/bg-mapping.avif)

## mapping is where the art is

what do you _do_ with the random number?

pitch, position, colour, size, timing...

choosing where **not** to be random matters just as much

---

{/* _class: impact */}

the balance between **predictability** and **surprise**

that's the craft

---

## re-creating static

```js
loadPixels();
for (let i = 0; i < pixels.length; i++) {
  pixels[i] = random() < 0.5 ? 0 : 255;
}
updatePixels();
```
